Struktur folder untuk Claude Code

HTML/CSS/JS murni — tanpa framework, siap dipakai sebagai starting point
rivulet-studio/
├── index.html        # Home
├── layanan.html       # Build & Craft pillars
├── portfolio.html     # Grid project
├── portfolio/
│  ├── project-1.html  # Detail per-project
│  ├── project-2.html
│  └── ...
├── tentang.html
├── kontak.html
│
├── css/
│  ├── variables.css    # Warna, font, spacing (:root)
│  ├── base.css        # Reset, typography dasar
│  ├── components.css   # Navbar, Button, Card, CTABand
│  └── pages/        # CSS khusus per-halaman (jika perlu)
│
├── js/
│  ├── components.js    # Navbar inject, footer inject (reuse HTML)
│  ├── animations.js    # GSAP ScrollTrigger setup
│  └── main.js         # Init umum, smooth scroll (Lenis)
│
└── images/
    ├── portfolio/
    └── icons/
💡 Catatan untuk Claude Code: karena tanpa framework, navbar & footer akan terduplikasi di tiap file .html. Dua cara mengatasi ini — minta Claude Code untuk:
(a) inject HTML navbar/footer lewat js/components.js (fetch partial HTML ke setiap halaman), atau
(b) simpan 1 file master lalu copy-paste manual ke tiap halaman saat ada perubahan.
Opsi (a) lebih rapi untuk maintain jangka panjang meski sedikit lebih rumit di awal.
